AmiBroker 32-bit vs 64-bit Compatibility Chart

SUMMARY

The following table clearly shows that AmiBroker Professional works on any Windows version starting from version 7 with every data plugin. 64-bit version of AmiBroker runs solely on 64-bit versions of Windows and only with a limited number of data sources due to lack of data vendors' API support for 64-bit technology.
