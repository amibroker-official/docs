Error 65. Internet handle passed to the Internet* function is invalid

You attempted to pass invalid Internet handle as an argument to InternetClose, InternetReadString functions. Internet* functions only operate on valid internet handles that are returned by InternetOpenURL function or InternetPostRequest function.

fh = fopen("localfile.txt", "w" );

if( fh )
{
    InternetReadString( fh ); // would generate error 65 as fh is NOT Internet handle

    InternetClose( fh ); // would generate error 65 as fh is NOT Internet handle

    fclose( fh ); // local file handle should be closed by fclose
}
