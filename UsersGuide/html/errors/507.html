Warning 507. Internet error ...

Your formula is using Internet functions and these internet function(s) caused run-time error such as connectivity error, etc. Detailed information about error is included in the warning text and it can be also retrieved in runtime using GetLastOSError() function.

ih = InternetOpenURL( "http://nonexistinghost.net" ); // would produce Warning 507. Internet error: The server name or address could not be resolved

if( ih )
{
   InternetClose( ih );
}
else
{
   printf("Something went wrong, OS error: %s\n", GetLastOSError() );
}
