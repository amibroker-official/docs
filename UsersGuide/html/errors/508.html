Warning 508. The same category is used multiple times in the `Filter settings` window. `GetOption()` can only return very first one.

Your formula is using GetOption() with one of the following fields:

"FilterIncludeIndex", "FilterIncludeFavorite", "FilterIncludeMarket", "FilterIncludeGroup", "FilterIncludeSector", "FilterIncludeIndustry", "FilterIncludeWatchlist", " FilterExcludeIndex", "FilterExcludeFavorite", "FilterExcludeMarket", "FilterExcludeGroup", "FilterExcludeSector", "FilterExcludeIndustry", "FilterExcludeWatchlist"

but the same category is selected more than once in Filter settings window. In such case GetOption would only return value of VERY FIRST category of selected type.

GetOption("FilterIncldueWatchlist") ; // will produce warning 508 if more than one watch list is selected in the Filter window.
