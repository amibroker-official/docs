Error 70. User agent can only be set once and only before opening any internet connections

You attempted to call InternetSetAgent twice or after opening Internet connection. That is incorrect. InternetSetAgent can only be called once and before opening any connections.

Wrong:

ih = InternetOpenURL("http://www.amibroker.com");

if( ih )
{
   InternetSetAgent("MyAgent"); // WRONG InternetSetAgent must be called BEFORE opening connection
   InternetClose( ih );
}


Correct:

InternetSetAgent("MyAgent"); // CORRECT InternetSetAgent must be called BEFORE opening connection

ih = InternetOpenURL("http://www.amibroker.com");

if( ih )
{
    InternetClose( ih );
}
