Warning 506. You have specified precision that exceeds IEEE standard. Numbers are only accurate upto the 7th significant digit.

Your formula is calling NumToStr() (WriteVal()) with precision exceeding IEEE standard. In 32-bit IEEE numbers only first 7 significant digits are guaranteed to be precise. You can avoid this warning by passing True as 4th argument (roundAndPad) in NumToStr / WriteVal() function call.

Example:

x = NumToStr( Volume, 20.0 ); // would produce Warning 506

y = NumToStr( Volume, 20.0, True, True ); // prevent Warning 506
