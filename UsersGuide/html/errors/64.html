Error 64. You have N Internet file handles (InternetOpenURL) that you forgot to close. You must call InternetClose function for every file opened with InternetOpenURL

You have called InternetOpenURL function but you forgot to call InternetClose to close the internet handle returned by InternetOpenURL:

Wrong:

InternetOpenURL("http://www.amibroker.com" );
// missing InternetClose would trigger Error 64

Correct:

ih = InternetOpenURL("http://www.amibroker.com" );

if( ih )
{
    InternetClose( ih ); // close internet handle properly
}
