DayOfWeek
- day of week

Date/Time
(AmiBroker 3.40)

SYNTAX dayofweek()
RETURNS ARRAY
FUNCTION Returns the array with day of week (0-6):
0 - Sunday
1 - Monday
...
5 - Friday
6- Saturday
EXAMPLE buy = dayofweek() == 1; // buy on Monday
sell = dayofweek() == 5; // sell on Friday
