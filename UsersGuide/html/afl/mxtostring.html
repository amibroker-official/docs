MxToString
- convert matrix to string

Matrix functions
(AmiBroker 6.10)

SYNTAX MxToString( mx )
RETURNS String
FUNCTION Creates string out of matrix variable in the Wolfram list style like this (for 3x3 matrix): "{ { x00, x01, x02 }, { x10, x11, x12 }, { x20, x21, x22 } }"
