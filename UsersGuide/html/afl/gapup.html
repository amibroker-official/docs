GapUp
- gap up

Basic price pattern detection

SYNTAX GapUp()
RETURNS ARRAY
FUNCTION Gives a "1" or "true" on the day a security's prices gap up. Otherwise the result is "0". A gap up occurs if yesterday's high is less than today's low.
