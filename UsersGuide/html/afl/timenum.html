TimeNum
- get current bar time

Date/Time
(AmiBroker 40)

SYNTAX TimeNum()
RETURNS ARRAY
FUNCTION Returns the array with numbers that represent quotation time coded as follows:
10000 * hour + 100 * minute + second, so 12:37:15 becomes 123715
