floor
- floor value

Math functions

SYNTAX floor( NUMBER )
floor( ARRAY )
RETURNS NUMBER,
ARRAY
FUNCTION Calculates the highest integer that is less than NUMBER or ARRAY.
EXAMPLE The function floor( 18.9 ) returns 18.
The formula floor( -19.9 ) returns -20.
SEE ALSO CEIL() function
