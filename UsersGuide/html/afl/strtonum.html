StrToNum
- convert string to number

String manipulation
(AmiBroker 4.50)

SYNTAX StrToNum( string )
RETURNS NUMBER
FUNCTION Converts string to number.
EXAMPLE List = "123,456,789";

for( i = 0; ( Item = StrExtract( List, i ) ) != ""; i++ )
{
   printf( "%gn", StrToNum( Item ) );
}
