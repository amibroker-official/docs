StrToLower
- convert to lowercase

String manipulation
(AmiBroker 4.80)

SYNTAX StrToLower( ''string'' )
RETURNS STRING
FUNCTION This function converts input string to all lower case.
EXAMPLE Title = StrToLower( "MiXeD CaSe" );
