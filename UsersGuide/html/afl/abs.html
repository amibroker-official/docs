abs
- absolute value

Math functions

SYNTAX abs( NUMBER )
abs( ARRAY )
RETURNS NUMBER
ARRAY
FUNCTION Calculates the absolute value of the NUMBER or ARRAY.
EXAMPLE The formula "abs( -15 )" will return +15; the formula "abs( 15)" also returns +15.
