sqrt
- square root

Math functions

SYNTAX sqrt( NUMBER )
sqrt( ARRAY )
RETURNS NUMBER,
ARRAY
FUNCTION Calculates the square root of NUMBER or ARRAY. The square root of a negative number always returns a zero result.
EXAMPLE The formula "sqrt( 16 )" returns 4
