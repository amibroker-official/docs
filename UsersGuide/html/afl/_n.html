_N
- no text output

Exploration / Indicators
(AmiBroker 4.10)

SYNTAX _N( string )
RETURNS nothing
FUNCTION Protects from printing string to the commentary output window
EXAMPLE _N( ticker = name() ); // thanks to _N function ticker symbol is not printed
SEE ALSO ENABLETEXTOUTPUT() function
