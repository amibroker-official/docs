Chr
- get string with given ASCII code

String manipulation
(AmiBroker 6.40)

SYNTAX Chr( code )
RETURNS STRING
FUNCTION The function returns string representing single character of given ASCII code.
EXAMPLE Chr(32); // will return " " (a string consisting of single space)
