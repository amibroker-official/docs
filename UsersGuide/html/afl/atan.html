Math functions

SYNTAX atan( NUMBER ),
atan( ARRAY )
RETURNS NUMBER
ARRAY
FUNCTION Returns the arc tangent of NUMBER or ARRAY. The value is returned in radians
EXAMPLE The formula "atan( 1.00 )" returns PI/4
