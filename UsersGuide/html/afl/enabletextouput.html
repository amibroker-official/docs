ENABLETEXTOUPUT
- enables/disables text output in the Chart Commentary window

Miscellaneous functions
(AFL 2.2)

SYNTAX EnableTextOuput( enable )
RETURNS NOTHING
FUNCTION Allows to enable or disable text output in the guru chart commentary window
EXAMPLE EnableTextOutput(False);
variable = "text"; // this won't be written to commentary window
EnableTextOutput(True);
SEE ALSO _N() function
