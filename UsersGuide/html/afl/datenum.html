DateNum
- date number

Date/Time
(AmiBroker 3.40)

SYNTAX datenum()
RETURNS ARRAY
FUNCTION Returns the array with numbers that represent quotation dates coded as follows:
10000 * (year - 1900) + 100 * month + day, so 2001-12-31 becomes 1011231 and 1995-12-31 becomes 951231
EXAMPLE datenum();
