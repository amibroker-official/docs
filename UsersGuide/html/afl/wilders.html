Wilders
- Wilder's smoothing

Moving averages, summation
(AmiBroker 3.40)

SYNTAX Wilders( ARRAY, periods )
RETURNS ARRAY
FUNCTION Calculates Wilder's average of the ARRAY using periods averaging range
EXAMPLE wilders( close, 10 );
