RWI
- random walk index

Indicators

SYNTAX rwi( minperiods, maxperiods )
RETURNS ARRAY
FUNCTION Calculates the Random Walk Index indicator as a difference between Random Walk Index from Highs (RWIHI() function) and Random Walk Index from Lows (RWILO() function.
EXAMPLE rwi( 9, 40 );
