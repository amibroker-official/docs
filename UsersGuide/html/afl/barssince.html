BarsSince
- bars since

Trading system toolbox

SYNTAX BarsSince( ARRAY )
RETURNS ARRAY
FUNCTION Calculates the number of bars (time periods) that have passed since ARRAY was true (or 1)
EXAMPLE barssince( macd() < 0 )
SEE ALSO BarsSinceCompare() function
