fclose
- close a file

File Input/Output functions
(AmiBroker 4.50)

SYNTAX fclose( filehandle )
RETURNS NOTHING
FUNCTION Closes a file.

The filehandle (NUMBER) should be the handle returned by fopen function.

EXAMPLE fh = fopen( "myfile.txt", "w");
if( fh )
{
   fputs( "Testing", fh );
   fclose( fh );
}
