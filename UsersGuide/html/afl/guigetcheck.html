GuiGetCheck
- get checked state of control

GUI functions
(AmiBroker 6.30)

SYNTAX GuiGetCheck( id )
RETURNS NUMBER
FUNCTION Get checked value of checkbox, toggle button or radio button. If control is checked (pressed) returns 1 (True), otherwise it returns 0 (False).

Parameters:

  • id - control ID
