AmiQuote's OLE Automation Object Model

Index of objects

  • Document

AmiQuote is an SDI (single document) application; therefore, there is only one class - Document - creatable using the following code:

JScript:

AB = new ActiveXObject("AmiQuote.Document");

VB/VBScript:

AB = CreateObject("AmiQuote.Document")

AFL:

AB = CreateObject("AmiQuote.Document");
