Functions accepting variable periods

The following functions support variable periods (where the periods parameter can be an array and changes from bar to bar):

  • AMA
  • AMA2
  • DEMA
  • HHV
  • HHVBars
  • IIR
  • LinRegSlope
  • LinearReg
  • LinRegIntercept
  • LLV
  • LLVBars
  • MA
  • PercentRank
  • Ref
  • StdErr
  • StDev
  • Sum
  • TEMA
  • TSF
  • WMA
