Keyboard shortcuts

AmiBroker allows complete customization of the user interface, including keyboard shortcuts. To define your own shortcuts, use the Tools->Customize menu, then select the Keyboard tab. Read more about it in the Tutorial: User Interface Customization.

The predefined keyboard shortcut list follows below. Please note that if you have used keyboard customization features, the list here may not be valid because some of the entries may have been customized by you.
