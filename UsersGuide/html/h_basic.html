Basic operations

Adding a new symbol

In order to add a new symbol into the database, you can use Symbol->New menu item or Add symbol toolbar button.

After selecting this function, you will be prompted for a new ticker symbol. The maximum ticker symbol length is 48 characters. For proper import functionality, you should enter the symbol in CAPITALS.

Removing a symbol

In order to remove an existing symbol from the database, you can use Symbol->Remove menu item or Remove symbol toolbar button. After choosing this function, you will be asked for confirmation of symbol removal. Note well that this operation cannot be undone !!!

Removing multiple symbols at once is possible using the Assignment organizer.

Splitting a stock

To perform a stock split, use Symbol->Split menu item or Split toolbar button.

AmiBroker provides an easy way of handling stock splits. The program will try to guess the split date and ratio by analyzing the quotations. If there is just a single quotation after the split, this should work; if not, you will be asked for the split date and ratio. Note well that this operation cannot be undone!!!

From version 2.0 and up, the split function offers more functionality: you can use the old-style ratio or you can specify a split using the following expression:

x->y

which means that x shares before the split become y after it. For example, 2->3 means that 2 shares become 3 after the split. So an ordinary split into five pieces will be 1->5.

As you have probably guessed, it is possible now to perform a reverse-split, for example 2->1, which means that 2 shares are joined together into one share.

Deleting a quotation

To delete a quotation, simply select the quote you want to delete by clicking on the chart (a vertical line will appear showing the selected date and quote). Then choose Edit->Delete quotation menu option.

To delete quotations of all stocks from a given day, you should use Edit->Delete session.

You can also use the Quote Editor to delete quotes.

Adding/Removing Symbols from Favorites

To add the symbol to your favorites, you should check the Favorites box in the Information window. To remove it from favorites, simply uncheck that box. Alternatively, you can click on the tree with the right-mouse button and select "Add to Favorites" and "Remove from Favorites" options from the context menu.

Merging quotations of two symbols

It sometimes happens that when a symbol's ticker is changed, you may get two tickers in your database - one holding historical quotes and the second one holding the latest quotes (after a name change). In order to put all quotes into a single ticker, you should use the Symbol->Merge feature. You should just select the new ticker (after the name change) and choose Symbol->Merge. Then from the combo, you should choose the original ticker ("merge with") and optionally check the following fields:

  • overwrite duplicate quotes - checking this option will overwrite the quotes already existing in the "new" ticker with those present in the "old" ticker (this should generally not be the case, but may happen).
  • delete "merge with" afterwards - checking this option will delete the "old" ticker after merging
  • assign alias name - checking this option will copy the "old" ticker to the alias field of the "new" ticker
