Layers context menu

The Layers context menu appears when you right-click on the layer list in the Layers tab of the Workspace pane.

Available options:

  • Add layer - Adds a new layer.
  • Remove layer - Removes the selected layer.

    Please note that you cannot remove the first 5 (built-in) layers.
  • Show All - Shows all unlocked layers.
  • Hide All - Hides all unlocked layers.
  • Toggle - Toggles the visibility of unlocked layers.
  • Lock built-in layers - Allows you to lock the first 5 (built-in) layers. When a layer is locked, its visibility changes automatically when the interval changes, and you cannot show/hide it manually.
  • Unlock built-in layers - Allows you to unlock the first 5 (built-in) layers. Once a layer is unlocked, its visibility does not change automatically when the interval changes, and you can show/hide it manually.
  • Properties -
    This launches a properties box that allows you to rename a layer and decide if a given layer should or should not be locked to the displayed interval. If you check the "Lock visibility to interval" box, the layer will show/hide automatically depending on what interval is

    currently displayed. You can define visibility for each layer using the "Interval" combo box and "Show/hide automatically" buttons. Note that there is a *separate* visibility setting for EACH interval. The layer properties box ALWAYS shows the "monthly" interval at start, but this is just a startup condition; you just switch to a particular interval
    and modify visibility.     To set up a locked layer completely, you have to set visibility for every layer listed in the "Interval" combo box. Simply select an interval and choose whether the layer should be shown or hidden for this interval. Select the next interval and again choose to show or hide. Select the next, and so on, until you define visibility for all intervals.

 
More information about what layers are and how to use them is in the Tutorial: Using Layers section of the guide.
